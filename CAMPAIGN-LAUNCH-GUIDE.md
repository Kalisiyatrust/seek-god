## OPENAI_API_KEY (Required for AI Chat)

The AI Chat feature on the portal requires an OpenAI API key. To activate it:

1. Go to https://platform.openai.com/api-keys
2. Create an API key
3. Add it to Vercel:
   - Go to https://vercel.com → your seek-god project → Settings → Environment Variables
   - Add: `OPENAI_API_KEY` = your key
   - Redeploy the project
